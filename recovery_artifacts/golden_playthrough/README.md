# Golden Playthrough Harvest

These files were generated by `tools/harvest_original_playthrough.py` from the known-good EXE.
Use `*.stdout.normalized.txt` for stable diffs against rebuilt executables.

Normalization strips ANSI, normalizes newlines, collapses long blank runs, replaces absolute Windows quicksave / slot paths with stable placeholders, then removes the nondeterministic VGA boot and ASCII title dissolve up to the stable `18+ ADULT CONTENT` age gate (so splash RNG does not affect diffs).

Main passes (`--mode all`):

- `global`: common panels and help/status surfaces.
- `rooms`: route/look/exits/nearby/scan/loot for every recovered room slug.
- `npcs_merchants`: NPC greeting/topic attempts and merchant wares/buy/sell attempts.
- `items`: find/examine/compare for recovered world item ids.

Optional: `--mode character` and `--mode surfaces` harvest additional scripts; parity with the golden EXE for those is still being aligned (turn counter / CC loop vs golden).

`tools/compare_playthrough_to_golden.py` copies harvested `*.save*.txt` from this folder into the diff output directory before each run so load-heavy scripts match the golden capture when present.
